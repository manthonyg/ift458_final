About this App

An application to allow management of personal loans and calculate the present value (PV) of the loan to a lender.

Application uses JWT authentication and will only allow a logged in user to view loans that are associated with them. This limits access to only the responsible party for the loan and is enforced by JWT session and a user id tied to each loan created.
